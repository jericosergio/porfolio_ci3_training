## üîß Troubleshooting

### Sidebar Won't Collapse?
**Solution**: Refresh the page or check browser console (F12)

### Layout Looks Broken on Mobile?
**Solution**: Make sure you're viewing in mobile-responsive mode
- Mobile devices: Works automatically
- Desktop browser: Open DevTools (F12) ‚Üí Toggle device toolbar

### Forms Not Submitting?
**Solution**: Check for red validation error messages above fields

### Alerts Not Showing?
**Solution**: Check the top of the page below navbar - alerts appear there

### Sidebar State Not Saving?
**Solution**: Check if cookies/storage are enabled in browser settings
